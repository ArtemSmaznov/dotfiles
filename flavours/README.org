* Dependencies
|----------+------------------------------|
| Name     | Description                  |
|----------+------------------------------|
| [[https://aur.archlinux.org/packages/flavours][flavours]] | A manager/builder for Base16 |
|----------+------------------------------|

#+begin_example shell
paru -S flavours
#+end_example

Before using =flavours= will need to pull all the templates and schemes
#+begin_example shell
flavours update all
#+end_example
